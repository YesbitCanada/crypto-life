# 币圈浮生记 — 玩家帮助手册

> 版本：v1.2.2 · 单文件 HTML5 游戏 · 无需注册，无需网络

---

## 故事背景

2023年，老同学小王说缅北有个"互联网客服"岗位，月入三万，包吃包住。"这是机会！"

你把房子抵了贷款凑路费，跟着去了。下飞机才发现不对劲——手机没收，护照扣押，签的是电诈合同。

三个月后，你和小王密谋出逃。翻墙、躲追捕、差点被噶腰子……好不容易跑回国内。

回来一查：**房子已经在走拍卖程序了。**

小王说"帮你东山再起"，借给你 **$3,000**：*"$2,000 帮你配了专业炒币电脑和硬件钱包，剩下 $1,000 就是你的启动资金。"*

（那台电脑，你至今没见到）

现在你欠他 **$3,000**，日息 **10%**。*"20天还清，不然去堵你。"*

**目标：在 20 天内翻盘还清债务，净资产归零或盈余即为胜利。**

---

## 胜负条件

| 结果 | 条件 |
|------|------|
| ✅ 胜利 | 心态值 > 0，且 第20天结束时 现金 + 存款 − 负债 ≥ 0 |
| ❌ 失败（心态崩溃） | 心态值归零 |
| ❌ 失败（债主上门） | 第20天结束净资产为负 |

## 界面说明

### 顶栏
- **现金**：当前钱包余额（USDT）
- **负债 / 日息10%**：当前欠小王的钱，每天自动 ×1.1
- **心态值**：归零即游戏失败
- **天数**：当前第几天 / 总20天
- **右侧警报框**：有爆仓风险时显示红色警报；平时显示最新事件摘要

#### 底部操作
- **下一天**：推进一天，触发价格变动和随机事件（这里展示“下一天”实际按钮，是推进游戏的核心功能）
- **全部平仓**：一键卖出所有持仓

### 左栏 — 信息源
- **4个信息源按钮**（Twitter / Discord / 交易所公告 / KOL）：点击切换，刷新内幕情报内容，纯氛围，不影响游戏数值
- **内幕情报卡**：每天随机更新，提供（不一定准确的）市场信号
- **今日情绪屏**：显示当天市场情绪，颜色随事件变化
  - `按兵不动` — 无事发生
  - `冲！冲！冲！` — 某币暴涨3倍以上
  - `恐慌踩踏` — 某币暴跌50%以上 / 债务追加
  - `钱包爆了` — 你的现金被钓鱼/冻结
  - `天降横财` — 获得免费空投
- **功能按钮**：AAVE存款 / 还款小王 / 心理援助 / Etherscan

### 中栏 — 市场与交易

#### 市场行情列表
每行显示一种资产：
- **图标**：左上角绿色 ● 表示当前持有该资产
- **代码 / 名称**
- **当前价格**
- **涨跌幅**（与上一天比较）
- **sparkline 折线图**：近10天价格走势

点击一行即选中该资产，开始交易。

#### 当前选择（左卡）
选中资产后显示：
- 币种代码 / 当前价格
- 持仓数量 / 爆仓价格
- 当前杠杆倍数 / 爆仓风险百分比

#### 交易控制（右卡）
- **数量滑条**：拖动选择交易手数，实时显示数量
- **杠杆选择**：1x（现货）或 2x（杠杆）
  - 已有持仓时，杠杆锁定为当前持仓杠杆，无法更改
- **买入**（绿色）：现金不足时自动置灰禁用
- **卖出**（红色）：无持仓时自动置灰禁用



### 右栏 — 账户与事件

#### 账户情况
- **持仓市值 / 持仓数量**：当前所有持仓的总市值和总手数
- **总持仓 / 仓位限制**：如 `35 / 100`，上限100手

#### 事件流
最近3条事件的卡片，显示图标、标题和描述。

#### 走势大图
选中资产的近10天价格折线图；有2x杠杆持仓时，红色虚线标注爆仓价。

## 交易机制

### 买入
- 花费 = 价格 ÷ 杠杆（每手保证金）
- 现金扣减，持仓增加
- 仓位上限 **100手**，超出无法买入

### 卖出
- 收益 = (当前价 − 入场价 × (1 − 1/杠杆)) × 手数
- 1x 杠杆卖出不会亏损本金（但价格跌了就少赚）
- 2x 杠杆卖出若价格低于入场价，收益可能为0（保证金损失）

### 杠杆与爆仓
| 杠杆 | 爆仓条件 | 爆仓后果 |
|------|----------|----------|
| 1x | 不爆仓 | — |
| 2x | 价格下跌 ≥ 50% | 持仓清零，心态值 −15 |

**同一币种只能持有一种杠杆**（锁定机制）。卖清后可重新选择杠杆再买。

## 功能说明

### 🏦 AAVE 存款
- 将现金存入 AAVE，**每天获得 1% 利息**（复利）
- 存款不受市场事件影响（钓鱼攻击只损失钱包现金）
- 两个独立滑条分别控制存入 / 取出金额
- "全部" 按钮快速填满滑条至最大值

### 💸 还款小王
- 输入金额还款，减少负债
- 负债利息每天10%复利，越早还越划算
- "全部还" 按钮快速还清当前最大可还金额

### 🧠 心理援助
- 花费 **$5,000** 将心态值恢复至 100
- 心态值归零 = 游戏失败，务必及时疏导

### 🔍 Etherscan
- 查看本局所有历史事件日志

## 随机事件系统

每天推进时，有 **65% 概率** 触发一条随机事件：

| 事件类型 | 效果 |
|----------|------|
| 价格暴涨 | 某资产价格暴涨很多倍 |
| 价格暴跌 | 某资产价格暴跌很多倍 |
| 免费空投 | 获赠若干手某资产（无需花钱） |
| 钱包攻击 | 现金损失一定比例，心态值下降 |
| 债务追加 | 小王追加 $500 债务 |

> 详细事件列表见 `event.md`，可自行修改后同步到代码。（这里不要显示在html里）

## 新手策略建议

1. **先还一部分债**：10%复利很恐怖，5天内最好就能还清债务，可以显著减少利息总额
6. **初期炒小币种**：初始资金有限，炒单价低的币，后期因为仓位有上限（100），炒贵的币种比如BTC
2. **用 AAVE 保住本金**：把部分现金存入 AAVE，免受钓鱼攻击，还能获得1%利息
6. **关注新闻热点**：币圈一天人间一年
4. **杠杆高风险高回报**：可能直接归零，也可能 ×2。杠杆爆仓需谨慎
6. **心态值低于 30 立刻去心理援助**：$5,000 换一条命
