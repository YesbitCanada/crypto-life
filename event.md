# 币圈浮生记 — 事件表

> 修改此文件后，需同步更新 `index.html` 中的 `EVENTS` 数组。
> 每天**必然**触发一条事件：**70%** 概率涨跌类，**20%** 概率心态类，**5%** 概率空投类，**5%** 概率债务类。

---

## 事件字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `title` | string | 事件标题（显示在弹窗标题栏和事件流） |
| `icon` | emoji | 事件头像 |
| `type` | enum | 事件类型：`price` / `goods` / `cash` / `debt` |
| `goods` | string | 影响的币种（`price`/`goods` 类型必填） |
| `rate` | number | 价格倍率（`price` 类型）：>1 涨，<1 跌 |
| `qty` | number | 免费获赠数量（`goods` 类型） |
| `hp` | number | 心态值变化（`cash` 类型，负数扣除） |
| `debt` | number | 追加债务金额（`debt` 类型） |
| `desc` | string | 事件描述正文 |
