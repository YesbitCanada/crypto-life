## 一、产品概述

### 背景
基于"北京浮生记"改编的单文件 HTML5 币圈题材经营模拟游戏。
玩家扮演被老同学小王拉入 PLUS 传销骗局的普通人，欠债 ¥3,000（日息10%），手持 ¥1,000 启动资金，20天内在币圈翻盘还债。

### 核心卖点
- 黑色幽默讽刺风格
- 币圈真实事件还原（DEFI崩盘、PLUS跑路、SEC起诉、马斯克推特）
- 杠杆爆仓机制带来的高风险快感
- 单文件部署，无需后端

### 目标文件
`crypto.html` — 新版像素风实现
参考文件：`crypto-cex.html`（当前暗色HUD版）、`crypto_life_pixel_ui.html`（像素UI原型）

## 二、游戏核心机制

### 2.1 初始状态
| 参数 | 值 |
|------|----|
| 初始现金 | ¥1,000 |
| 初始负债 | ¥3,000 |
| 日利息 | 10%（复利） |
| 游戏天数 | 20天 |
| 初始仓位上限 | 100手 |
| 初始心态值 | 100 |

### 2.2 交易资产
| 代号 | 名称 | 价格区间 | 颜色 |
|------|------|----------|------|
| BTC | 比特币 | 8000–126,000 | #f7931a |
| ETH | 以太坊 | 200–5,000 | #627eea |
| DOGE | 狗狗币 | 0.5–40 | #c2a633 |
| GPU | ETH矿机显卡 | 1000–9000 | #ff4500 |
| MINER | BTC矿机 | 100–2,000 | #0052ff |
| NFT | 总统代言 | 3000–18,000 | #9b59b6 |
| LP | DeFi矿池 | 80–2,000 | #00c896 |
| PLUS | PLUS传销币 | 10–500 | #666666 |

### 2.3 杠杆机制
- 可选倍数：**1x / 2x**（最大2x，不开放5x/10x）
- 保证金 = 价格 ÷ 杠杆
- 爆仓价 = 入场价 × (1 - 1/杠杆)
- 2x杠杆：价格下跌50%触发爆仓
- 爆仓后：持仓清零，心理值 -15

### 2.4 每日流程（doMove）
1. 天数 +1，剩余天数 -1
2. 负债 × 1.1（复利）
3. 所有资产价格随机重置（区间内）
4. 检查爆仓（checkLiquidations）
5. 65% 概率触发随机事件
6. 更新UI，弹出事件弹窗

### 2.5 胜负判定
- **胜利**：心态值 > 0 且 (现金 + 存款 - 负债) ≥ 0
- **失败1**：心态值归零（心态崩溃）
- **失败2**：20天到期净资产为负

## 三、UI 需求（新版像素风）

### 3.1 整体风格
- **参考**：`crypto_life_pixel_ui.html`
- 像素边框：所有面板 4px solid `#2b241d` + inset 内阴影（立体效果）
- 主面板底色：羊皮纸 `#e7d8b1`
- 电子屏：黑色背景 `#0a0f14` + 绿色荧光数字 `#6cff3d`
- 字体：`Courier New` 等宽
- 按钮：像素街机风，有按压动效

### 3.2 顶栏（Topbar）
```
[ 币圈浮生记 ]  [ 现金 | 负债（日利息） | 心态 | 天数 ]  [ ⚠ 风险警报 ]
  蓝色渐变框      4格状态栏                       红色警报框
```
- 状态栏：电子屏风格显示数值
- 警报框：有爆仓风险时显示，无风险时显示最新事件简报

### 3.3 左栏（信息源）
- **信息源按钮**（4个，像素按钮，可选中高亮）
  - 🐦 Twitter 传闻
  - 💬 Discord 小道
  - 📊 交易所公告
  - 👤 KOL 观点
- **内幕情报卡**：每天随机更新一条内幕消息
- **今日情绪屏**：电子屏显示当日情绪词（FOMO / PANIC / GREED / HOLD）
- **功能入口**（底部，像素按钮）
  - 🏦 AAVE存款
  - 💸 还款小王
  - 🧠 心理援助
  - 🔍 Etherscan

### 3.4 中栏（市场 + 交易）
#### 市场行情列表
列：币种图标 | 名称 | 价格 | 涨跌% | 迷你折线图（黑屏SVG）

行状态：
- 默认：羊皮纸底
- 选中：蓝色高亮（同原型 `.coin-row.active`）
- 持仓中：左侧有绿色竖边标记

#### 交易控制（2卡片网格）
**左卡：当前选择**
- 电子屏：币种名 / 当前价格 / 持仓手数

**右卡：交易控制**
- 数量输入框（数字，非滑块）
- 杠杆下拉框（1x / 2x）
- [买入] 绿色 / [卖出] 红色 大按钮

#### 底部操作栏
- [⏩ 下一天] 黄色全宽按钮（占1/2）
- [全部平仓] 灰色按钮（占1/2）

### 3.5 右栏（账户 + 事件）
#### 账户电子屏（3块）
- 总资产（持仓市值）
- 已开杠杆（显示当前选择的杠杆倍数）
- 爆仓风险%（距爆仓的价格空间百分比）

#### 事件流（近3条，卡片式）
每条格式：
```
[ 😤 头像 ] | 事件标题（加粗）
             | 事件描述文字
```

#### 走势大图（底部）
- 选中币种的价格历史折线
- 像素风黑屏SVG，方角折线（`stroke-linecap: square`）

## 四、事件系统

### 4.1 事件类型
| 类型 | 效果 | 示例 |
|------|------|------|
| price | 某币价格 × rate | PLUS崩盘 × 0.01 |
| goods | 免费获得N手 | 空投5手SHIB |
| cash | 现金 × rate，心态 +hp | 钓鱼攻击损失40% |
| debt | 负债追加 | 小王追加5000债务 |

### 4.2 事件触发
- 每天 65% 概率触发1个随机事件
- 爆仓优先弹窗，之后再弹事件弹窗（链式回调）
- 事件结果写入事件流（右栏）

### 4.3 今日情绪映射
| 条件 | 情绪词 |
|------|--------|
| 无事件 | HOLD |
| price事件涨幅 > 3x | FOMO |
| price事件跌幅 > 50% | PANIC |
| cash事件（损失） | REKT |
| goods事件（空投） | MOON |

### 4.5 特殊事件

PLUS代币如果遇到跑路事件，价格区间改为0.01-0.1浮动。
